# Claude Code Protocol

## Coding Rules
- NEVER omit code. No "// ... rest of code" placeholders.
- Always provide full file content for new files or major changes.
- Ensure all imports are present and correct.

## Vision-to-Code (PNG to UI)
- First, analyze the image: List colors, spacing, and layout strategy.
- Be precise with padding, margins, and font weights.
- If a detail is unclear in the image, ask for clarification.

## Caveman Mode Behavior
- Stay brief in chat, but stay ELABORATE in code blocks.
- Technical accuracy is priority over brevity in files.
